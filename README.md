# Codex Skills

This repository publishes a subset of local Codex skills.

## Published skills

- `mermaid-github-safe-area`: Ensure Mermaid diagrams in GitHub READMEs reserve bottom-right safe area using init-only config (no CSS, no spacer nodes) and optionally apply handrawn style on request.
- `waylog-journal`: Summarize `.waylog/history/*.md` into a sanitized per-session cache + a condensed, chronological `.waylog-journal/summary.md` journal. Use WayLog CLI (`waylog`) to populate `.waylog/history/*.md` (`https://github.com/shayne-snap/waylog-cli`).
