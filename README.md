# Codex Skills

This repository publishes a subset of local Codex skills.

## Published skills

- [`mermaid-github-safe-area`](mermaid-github-safe-area): Ensure Mermaid diagrams in GitHub READMEs reserve bottom-right safe area using init-only config (no CSS, no spacer nodes) and optionally apply handrawn style on request.
- [`waylog-journal`](waylog-journal): Summarize `.waylog/history/*.md` into a sanitized per-session cache and a condensed, chronological `.waylog-journal/summary.md` journal. Use WayLog CLI ([`waylog`](https://github.com/shayne-snap/waylog-cli)) to populate `.waylog/history/*.md`.
